# Command Palette

The Command Palette is like a search bar that can do almost anything in FXFlow. Instead of clicking through menus, you just type what you want and it takes you there. Think of it like Spotlight on a Mac or the search bar on your phone's home screen.

## Opening the Command Palette

There are two ways to open it:

- **On Mac:** Press **Cmd + K** (hold the Command key and press K)
- **On Windows/Linux:** Press **Ctrl + K** (hold the Control key and press K)

A search box will pop up in the center of your screen with a text field where you can start typing right away.

> [!TIP]
> This is one of the fastest ways to get around FXFlow. Power users rely on the Command Palette instead of clicking through the sidebar. Once you get used to it, it feels like a superpower.

## How to Use It

1. Open the Command Palette with the keyboard shortcut
2. Start typing what you're looking for (e.g., "charts" or "settings")
3. Results appear instantly as you type — no need to press Enter first
4. Use the **arrow keys** (up/down) to highlight the result you want
5. Press **Enter** to select it

That's it! The palette closes and you're taken to the page or the action runs.

## What You Can Find

The Command Palette organizes results into sections:

### Navigate

This section lets you jump to any page in FXFlow. Type a page name and it shows up:

- Dashboard
- Positions
- Charts
- Analytics
- Trade Finder
- TV Alerts
- AI Analysis
- AI Trader
- Settings
- ...and more

You don't have to type the full name. Typing "pos" will find "Positions." Typing "set" will find "Settings."

### Actions

These are quick things you can do without navigating to a full page first. Actions let you trigger common tasks right from the palette.

### Quick Settings

Some settings can be toggled on or off directly from the Command Palette — no need to open the Settings page. This is handy for things you change often.

> [!NOTE]
> The results are "fuzzy matched," which means you don't have to spell things perfectly. If you type "trad find," it will still match "Trade Finder." Pretty smart!

## Closing the Command Palette

There are several ways to close it without doing anything:

- Press the **Escape** key
- Click anywhere outside the palette
- Press **Cmd + K** (or **Ctrl + K**) again

The palette disappears and you're right back where you were.

## Tips for Getting the Most Out of It

- **Be brief** — Type just 2-3 characters and the results usually narrow down enough to find what you need.
- **Use it for speed** — Navigating with the Command Palette is almost always faster than clicking through the sidebar, especially once you build the muscle memory for Cmd+K.
- **Works everywhere** — You can open the Command Palette from any page in FXFlow. It doesn't matter where you are.

> Try it right now! Press Cmd+K (or Ctrl+K), type "dash," and press Enter. You should land on the Dashboard page.
