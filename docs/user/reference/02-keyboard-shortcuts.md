# Keyboard Shortcuts

FXFlow is designed to be fully usable with just a keyboard. Here are all the shortcuts you can use.

> [!TIP]
> The **Command Palette** is the fastest way to get anywhere in FXFlow. Press **Cmd + K** (Mac) or **Ctrl + K** (Windows/Linux) and start typing what you want to do.

---

## Global

| Shortcut               | What it does                                          |
| ---------------------- | ----------------------------------------------------- |
| **Cmd/Ctrl + K**       | Open the Command Palette                              |
| **Escape**             | Close any open dialog, sheet, or the Command Palette  |
| **Tab**                | Move forward through buttons, links, and form fields  |
| **Shift + Tab**        | Move backward through buttons, links, and form fields |
| **Enter** or **Space** | Activate the currently focused button or control      |

## Command Palette Navigation

Once the Command Palette is open, type to search, then use these keys:

| Shortcut            | What it does                     |
| ------------------- | -------------------------------- |
| **Arrow Up / Down** | Move through the list of results |
| **Enter**           | Run the selected command         |
| **Escape**          | Close the palette                |

## Quick Navigation (from Command Palette)

These shortcuts are shown inside the Command Palette and navigate you to a page instantly:

| Shortcut | Destination |
| -------- | ----------- |
| **G D**  | Dashboard   |
| **G P**  | Positions   |
| **G C**  | Charts      |
| **G S**  | Settings    |

> [!NOTE]
> You can also find **Trade Finder**, **TradingView Alerts**, **AI Analysis**, and **AI Trader** in the Command Palette by typing their names.

## Quick Settings (from Command Palette)

| Shortcut     | What it does                      |
| ------------ | --------------------------------- |
| **Ctrl + B** | Toggle the sidebar open or closed |

You can also switch between dark and light mode or jump to Trading Mode settings by searching in the Command Palette.

## Charts

| Action              | How to do it                      |
| ------------------- | --------------------------------- |
| Zoom in / out       | Scroll wheel or pinch on trackpad |
| Scroll through time | Click and drag left or right      |

> [!TIP]
> If you prefer the mouse, every shortcut action is also available through menus and buttons in the interface. Keyboard shortcuts are just a faster way to get there.
