# Code of Conduct

ForexFlow adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct. The full text is available at that link and applies to all project spaces — this repository, its issue tracker, discussions, pull requests, and any associated community channels.

## Scope

This Code of Conduct applies to everyone participating in ForexFlow — contributors, maintainers, and users — whenever they are acting on behalf of the project or in project spaces.

## Reporting

If you experience or witness behavior that violates this Code of Conduct, report it through GitHub's private reporting mechanism:

👉 **[Open a private security advisory](https://github.com/bmarshall511/forexflow/security/advisories/new)** — choose "Code of Conduct" as the report type.

Reports are read only by repository maintainers. You will receive an acknowledgement within 72 hours.

## Enforcement

Maintainers are responsible for clarifying and enforcing the standards described in the Contributor Covenant. Consequences for unacceptable behavior follow the [enforcement ladder](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines) described in the Covenant: correction, warning, temporary ban, permanent ban.
